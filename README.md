# homewizard-cli

High-performance CLI for HomeWizard P1 Meter.

## Installation

```bash
pip install homewizard-cli
```

## Usage

```bash
# Quick summary
homewizard-cli

# Full data dump
homewizard-cli data

# Power monitoring
homewizard-cli power

# Device info
homewizard-cli info
```

## Requirements

- Python 3.11+
- HomeWizard P1 Meter on local network
